# A few nice to have features/fixes for _Kritter_

_The features were tested in solo with keyboard + mouse (and rarely with a gamepad)_

## **Features**
- _Building_
  - Show green preview if one of the buildings can be built instead of just checking the first building of the list.

- _Cursor_
  - Add ingame crosshair
  - Add color setting for crosshair
  - Lock cursor in the window during gameplay

- _Ingame Menu_
  - Add a validation popup before leaving gamaeplay/game

- _Loot_
  - Auto loot items when the player can interact with them (doesn't work with potion and probably some special items)

## **Setup**

1. Get BepInEx release **https://github.com/BepInEx/BepInEx** and install it to Kritter folder
2. Get the latest release of the mod **https://github.com/Tetex/KritterNiceToHaveMod/releases**
3. Copy it to **<Kritter_folder>/BepInEx/Plugins/**
4. Launch the game, it will create **KritterNiceToHavePlugin.cfg** in **<Kritter_folder>/BepInEx/config/**
5. You can now edit the cfg and toggle features
